# シェルの組み立て

##　Groveケーブルを通す。

シェルの横についている小窓からGroveケーブルを出します。

<img src="../pic/IMG20250314213604.jpg" width=640>

## シェルをサーボホーンに固定する

シェルの外側からM1.7ネジを使って4箇所固定します。
もし、固定がしづらい場合はサーボホーンを外してシェルに取り付けてからでも構いません。
